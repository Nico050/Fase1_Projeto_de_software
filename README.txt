Sobre o Programa
Bem-vindo ao repositório do Gerenciador de Tarefas e Projetos! Este programa foi desenvolvido para ajudar você a organizar suas tarefas e projetos de forma eficiente, com uma interface simples e intuitiva. Uma das maiores vantagens deste software é que ele é totalmente portátil: você só precisa baixar o executável para começar a usá-lo imediatamente, sem a necessidade de instalações complicadas ou dependências adicionais.

O programa utiliza um banco de dados local para armazenar informações de usuários, tarefas e projetos, garantindo que seus dados estejam sempre seguros e acessíveis diretamente no seu computador. Além disso, ele oferece uma ferramenta de login e cadastro segura, permitindo que você gerencie suas atividades de forma personalizada.

Funcionalidades Principais
1. Ferramenta de Login e Cadastro
Cadastro: Crie uma conta de usuário de forma rápida e segura. O programa verifica se o nome de usuário já está em uso, garantindo que cada conta seja única.

Login: Faça login com suas credenciais para acessar suas tarefas e projetos. O sistema valida o nome de usuário e senha, garantindo acesso seguro.

2. Gerenciamento de Tarefas
Criação de Tarefas: Adicione tarefas com nome, prazo e prioridade. Todas as tarefas são salvas no banco de dados local e podem ser visualizadas ou removidas a qualquer momento.

Remoção de Tarefas: Remova tarefas que já foram concluídas ou que não são mais necessárias.

3. Gerenciamento de Projetos
Criação de Projetos: Crie projetos com nome, prazo e prioridade. Adicione colaboradores ao projeto, que também poderão visualizá-lo em suas próprias contas.

Remoção de Projetos: Remova projetos e todas as suas referências dos colaboradores.

4. Banco de Dados Local
O programa cria e gerencia automaticamente um banco de dados local usando arquivos CSV. Cada usuário tem seu próprio arquivo de tarefas (u{username}.csv), e as informações de login são armazenadas em um arquivo central (users.csv).

Isso garante que seus dados estejam sempre seguros e acessíveis, sem a necessidade de conexão com a internet ou servidores externos.

5. Portabilidade
O programa é totalmente autossuficiente. Basta baixar o executável e rodá-lo. Não há necessidade de instalação ou configurações complexas. Tudo o que você precisa está em um único arquivo!

Como Usar
Download: Baixe o executável do programa.

Execução: Execute o arquivo baixado. O programa será iniciado automaticamente.

Cadastro/Login: Se você já possui uma conta, faça login. Caso contrário, crie uma nova conta usando a ferramenta de cadastro.

Gerenciamento de Tarefas e Projetos: Após o login, você pode adicionar, remover ou visualizar tarefas e projetos. Para projetos, você pode adicionar colaboradores, que também poderão visualizar o projeto em suas contas.

Sair ou Alterar Usuário: Você pode sair do programa ou alterar o usuário a qualquer momento.

Exemplo de Uso
Cadastro de Usuário:
Execute o programa.

Escolha a opção "Cadastro".

Insira um nome de usuário e senha.

Pronto! Sua conta foi criada.

Criação de Tarefa:
Faça login.

Escolha a opção "Adicionar Tarefa".

Insira o nome, prazo e prioridade da tarefa.

A tarefa será salva automaticamente no banco de dados local.

Criação de Projeto:
Faça login.

Escolha a opção "Adicionar Projeto".

Insira o nome, prazo e prioridade do projeto.

Adicione colaboradores ao projeto.

O projeto será salvo automaticamente no banco de dados local e estará disponível para os colaboradores.
